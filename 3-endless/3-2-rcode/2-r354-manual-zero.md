# 3.2.2 R354 엔드리스 Zero 실행
R350 코드에 의한 수동 제로 기능은 로봇이 정지하고 있을 때 프로그램 명령(endless zero)을 대신하여 수동 혹은 자동모드에서 사용자가 축 위치를 0deg로 설정하고자 할 때 사용합니다.

|         **R코드**     |         **파라미터**  |        **설명**       |
| :-------------------: | :-------------------: | :-------------------: |
| R354                  |        0              | 모든 축에 대한 Zero    |
| R354                  |   엔드리스 축 번호     | 지정한 축에 대한 Zero  |
